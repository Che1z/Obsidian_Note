#HTML 
 頁面中組成一個可見區塊，本身會佔據**一整行**，前後內容則會以換行方式分隔
 Block element可利用巢套方式套在其餘Block element中
 > Block element
 > -- Block element
 > ---- Block element (Nested)

Block element不可巢套在Inline element中。但Inline element可巢套在Block element中

> Inline element
> --Block element (不可行)

> Block element
> --Inline element (可行)

Block element有：
 - p tag (Paragraph)
 - ul、ol、li tag (List)
 - div (Division)
